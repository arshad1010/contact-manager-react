# Contact Manager
A simple contact manager to manage CRUD operations of contacts. Use save button to locally store any changes and comeback at any time to get all saved contacts.
## Functionalities
* Create Contact
* Read Contacts
* Update Contact
* Delete Contact
* Save Contact Locally
* Delete all Contacts
